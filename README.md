# Ryze Portfolio

Portfolio officiel de **Ryze**, monteur vidéo spécialisé en **Highlights Gaming**.

Site développé avec **React + Vite + Bootstrap**.  
Design moderne, responsive et optimisé pour desktop / mobile.

---

# Aperçu

Le site contient :

- Home page premium
- Page Highlights (gallery vidéos YouTube)
- Page Clients
- Mentions légales
- Multi-langue FR / EN
- Background animé gaming
- Responsive mobile

---

# Technologies utilisées

- React
- Vite
- React Router DOM
- React Bootstrap
- Bootstrap
- React Icons
- i18next
